# AI Customer Support Agent for Retail Mobile Store

## Project Overview
This project is an AI-powered customer support system built using Salesforce Agentforce and Service Cloud.

## Features
- Automatically responds to customer queries
- Handles mobile product-related issues
- Uses AI agent for smart responses
- Improves response time

## Tools & Technologies Used
- Salesforce Developer Org
- Agentforce
- Service Cloud
- Flow Builder
