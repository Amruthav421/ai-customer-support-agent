AI Customer Support Agent for Retail Mobile Store

 Project Overview
This project is an AI-powered Customer Support system built using Salesforce Developer Org.  
It automates customer queries for a retail mobile store using Salesforce CRM tools like Flows, Apex, and Agentforce concepts.

---

Objective
The main objective of this project is to:
- Automate customer support process
- Reduce manual work in handling cases
- Improve response time using AI-like logic
- Improve customer experience

---

 How It Works
1. Customer raises a query or issue
2. Salesforce Flow/Apex processes the request
3. System identifies correct action based on logic
4. Case is updated or resolved automatically
5. Customer gets response faster
